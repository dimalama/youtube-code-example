The project is about creating simple youtube videos search using backbonejs, requirejs and ejs templates.
It is expected user to see five search results on the page and if there are more then five videos has been found we should
organize the next/prev page rearch results view. 

In order to run this project you need to use your local web server.

P.S. Please note that included libs which use this project are not minimized because it is development version.
